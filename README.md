# Violetta Portfolio

A React portfolio website for showcasing projects, skills, and contact information.

## Run locally

```bash
npm install
npm start
```

## Build for production

```bash
npm run build
```

## Deploy to Vercel

1. Upload this project to GitHub.
2. Import the GitHub repo into Vercel.
3. Use the default React settings:
   - Build command: `npm run build`
   - Output directory: `build`
